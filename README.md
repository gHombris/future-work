# Global Solution - Future of Work 

##  Resumo do Projeto
Este projeto foi desenvolvido como parte da avaliação **Global Solution - Front-End Design & Web Development**. 

O tema central é **"O Futuro do Trabalho"**. A aplicação consiste em uma plataforma web colaborativa (SPA), inspirada no LinkedIn, que conecta profissionais de diferentes áreas. O objetivo é permitir a visualização de perfis, competências e experiências, promovendo um ecossistema de tecnologia mais justo e inclusivo.

A aplicação consome um arquivo JSON local com 60 perfis simulados e oferece funcionalidades de busca, filtragem, visualização detalhada e alternância de tema (Dark Mode).

##  Tecnologias Utilizadas
* **React.js:** Biblioteca principal para construção da interface.
* **Tailwind CSS:** Framework de estilização utility-first.
* **Lucide React:** Biblioteca de ícones.
* **JavaScript (ES6+):** Lógica de manipulação de dados e eventos.

---

##  Funcionalidades
* **Listagem de Profissionais:** Exibição de cards com resumo do perfil.
* **Busca Inteligente:** Filtra profissionais por Nome, Cargo, Área, Cidade ou Habilidade Técnica.
* **Modal de Detalhes:** Visualização completa do perfil (Experiências, Formação, Soft Skills, Projetos).
* **Dark Mode:** Alternância entre tema claro e escuro persistente.
* **Responsividade:** Layout adaptável para Mobile, Tablet e Desktop.
